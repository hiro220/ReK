# Gitの使い方について

## clone
```
git clone https://github.com/hiroki-inoue-git/ReK
```

## checkout
```
git checkout [チェックアウト先のbranch名]
```

## commit
```
git commit -m "ここにコメントを入力する"
```

## push
```
git push origin [branch名]
```

## pull
```
git pull origin [branch名]
```

## merge
```
git merge [マージしたいbranch名]
```
